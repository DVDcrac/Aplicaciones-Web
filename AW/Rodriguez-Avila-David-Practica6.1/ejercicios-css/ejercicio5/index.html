Estilos CSS: ID

Dentro de la hoja puedes aplicar estilos a etiquetas muy específicas a través del atributo ID.

<.p.> etiqueta con ID de estilo diferente

En el archivo CSS, un estilo para ID se define con un nombre precedido por un pad, como por ejemplo: #definition { propiedad1: valor1; propiedad2: valor2; } Este es el párrafo con el estilo #special_paragraph especificado en la hoja de estilos. Se aplica con el atributo id.
